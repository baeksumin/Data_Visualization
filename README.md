## Data_Visualization



> 기간
 
2019.11-2019.12

> 프로젝트 간략 소개

간단한 실습으로 어떠한 그래프를 그리기 위해 r graphics와 ggplot2를 사용할 때 코드가 어떻게 다른지 비교함.

> 실행환경

R

---

### 프로젝트 구조

> 1. 데이터 소개

<img width="500" alt="스크린샷 2021-01-20 오후 11 44 36" src="https://user-images.githubusercontent.com/49911177/105191280-3ba8fb80-5b7a-11eb-8780-4f0d967e60e5.png">

- diamonds 데이터세트는 다이아몬드의 특성에 관한 데이터이다.
- 절단면의 등급이나 다이아몬드의 크기와 가격과의 상관관계나 또 어떤 종류의 다이아몬드가 많은지에 대해 시각화 해보면 데이터를 더 쉽게 이해할 수 있을 것이라고 생각하였다.

> 2. 점 그래프

<img width="500" alt="스크린샷 2021-01-20 오후 11 44 54" src="https://user-images.githubusercontent.com/49911177/105191287-3ea3ec00-5b7a-11eb-8eba-f585518ef606.png">

- 대각선으로 점이 분포되어 있는데, 이는 다이아몬드의 크기와 가격 간에 상관관계가 있다는 것을 알 수 있다.
- r그래픽스에서는 plot()함수를 사용하여 type인수에 p를 사용하였다.
- ggplot2에서는 geom_point()함수를 사용하였다.

> 3. 선 그래프

<img width="500" alt="스크린샷 2021-01-20 오후 11 45 08" src="https://user-images.githubusercontent.com/49911177/105191298-41064600-5b7a-11eb-862e-dafcbdc4de41.png">

- 점 그래프와 마찬가지로 두 변수 사이에 상관관계가 있다는 것을 알 수 있다.
- r그래픽스에서는 plot()함수를 사용하여 type인수에 l를 사용하였다.
- ggplot2에서는 geom_line()함수를 사용하였다.
- r그래픽스를 사용한 그래프는 데이터를 연결하는 선들이 ggplot2를 사용한 그래프와는 달리 가로로 이어져있다. 

> 4. 막대 그래프

<img width="500" alt="스크린샷 2021-01-20 오후 11 45 24" src="https://user-images.githubusercontent.com/49911177/105191309-44013680-5b7a-11eb-9a55-9ab2183fbfeb.png">

- 투명도에 관한 막대그래프이다.
- r그래픽스에서는 전체 데이터세트에서 투명도 변수만을 추출하여 테이블을 만든 후 barplot()함수를 사용하여 그래프를 그렸다.
- ggplot2에서는 geom_bar()함수를 사용하였다.

> 5. 누적 막대 그래프

<img width="500" alt="스크린샷 2021-01-20 오후 11 45 44" src="https://user-images.githubusercontent.com/49911177/105191313-45326380-5b7a-11eb-8b61-9b5170d04ac2.png">

- 위에 그렸던 막대 그래프에 cut 변수를 추가하여 누적 막대 그래프를 그렸다.
- r그래픽스에서는 막대 그래프에 색을 지정하지 않았더니 흑백으로 나타나서 하나하나 색을 지정해주었다.
- ggplot2에서는 색을 지정하지 않아도 "fill=cut" 이라고 쓰면 색이 저절로 지정되었다.
